How to Use

Steam & Standalone Version

These versions maintain the standard keybinds that Starblast players are already familiar with.

  • Steam Version: Requires your 64-bit Steam ID with Starblast activated on the account
  • Standalone Version: Requires a valid ECP Key

Browser Version

The browser version comes with special keybinds for an improved experience:

  • CTRL+X: Reloads the page (required to leave a game or for quick rage quitting)
  • CTRL+E: Opens in-game settings
  • CTRL+S: Quits Starblast and returns to the main menu of the application

Compatibility

Currently, the Starblast Launcher only works on Windows 10 & Windows 11.

As a Windows user, I cannot pack applications for other operating systems. If you are a node.js/electron developer on a different platform and would like to help, please get in contact with me (official_troller on Discord).
